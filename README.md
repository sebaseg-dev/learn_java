# Learning Java: From Zero to Spring

## Context

In January 2026, I earned my RNCP6 Concepteur Développeur d'Applications (Software Designer and Developer) degree from OpenClassrooms. During the final quarter of 2025, I independently developed a LegalTech SaaS. I leveraged my degree coursework and self-taught PHP/Symfony skills to build a full-stack application featuring a decoupled architecture (Symfony and React connected via a RESTful API) and a PostgreSQL database. Unfortunately, the project was shelved commercially at the end of the year following downturns in my target market.

Initially, I considered maintaining this software as a portfolio piece to showcase my skills to recruiters—viewing a salaried position as the most stable way to transition my career. However, continuing work on a product that will never be launched felt counterproductive. Furthermore, the roles I am most interested in often require a different set of technical skills.

For the past six months, I have felt the need to add a robust, enterprise-grade language (such as Java, Go, or Rust) to complete my stack. That is the journey I am starting today.

## Why Java?

At first, I considered Go or Rust. They are trendy, used in large-scale production, modern, and well-documented.

However, based on the job market data I have been scraping daily for weeks, those languages are rarely mentioned for junior or mid-level roles in my area.

I am looking for a position that aligns with my core values. I consistently see high-quality offers—often requiring more experience than I currently have—specifically seeking Java/Spring developers. Therefore, I have chosen to align my skill set with market demand.

## How?

I have a degree from OpenClassrooms. Although the courses aren't perfect, they are completely free, well-organized, and quick to complete. They structure their content into "paths," which is useful for organizing a self-learning curriculum. Based on the [Full-Stack Developer - Java and Angular](https://openclassrooms.com/fr/paths/2460-developpeur-full-stack-java-et-angular) path, I have designed my own as follows:

1. ✅ [Apprenez à utiliser la ligne de commande dans un terminal](https://openclassrooms.com/fr/courses/6173491-apprenez-a-utiliser-la-ligne-de-commande-dans-un-terminal)
2. ✅ [Découvrez le fonctionnement des algorithmes](https://openclassrooms.com/fr/courses/7527306-decouvrez-le-fonctionnement-des-algorithmes)
3. ~~[10h facile – Apprenez à programmer en Java (obsolete)](https://openclassrooms.com/fr/courses/6173501-apprenez-a-programmer-en-java)~~
4. [10h facile – Apprenez à programmer en Java (up-to-date)](https://openclassrooms.com/fr/courses/8383791-apprenez-a-programmer-en-java)
5. [4h facile – Écrivez du code Java maintenable avec MVC et SOLID](https://openclassrooms.com/fr/courses/6810956-ecrivez-du-code-java-maintenable-avec-mvc-et-solid)
6. ✅ [Adoptez les API REST pour vos projets web](https://openclassrooms.com/fr/courses/6573181-adoptez-les-api-rest-pour-vos-projets-web)
7. [6h moyenne – Implémentez vos bases de données relationnelles avec SQL](https://openclassrooms.com/fr/courses/6971126-implementez-vos-bases-de-donnees-relationnelles-avec-sql)

9. [10h moyenne – Testez votre code Java pour réaliser des applications de qualité](https://openclassrooms.com/fr/courses/6100311-testez-votre-code-java-pour-realiser-des-applications-de-qualite)

11. [10h difficile – Utilisez Spring Data pour interagir avec vos bases de données](https://openclassrooms.com/fr/courses/6982461-utilisez-spring-data-pour-interagir-avec-vos-bases-de-donnees)

In total, 54 hours are estimated. I know from experience that the duration can be overestimated for "easy" courses and underestimated for "difficult" ones.

My goal is to apply the concepts from these courses to a live project, building an application from scratch to a rough POC (Proof of Concept) by the end of the learning path.

## Project Idea

When I was looking for information on how to start a company in France and how to choose between different legal forms, I was overwhelmed—even with my legal, accounting, and tax background. It is incredibly complex to calculate how the choice of legal form impacts taxes, and official tools sometimes provide incorrect or undocumented results.

I would like to develop an open-source app to help entrepreneurs navigate these options. The project could evolve from a simple Java terminal program for calculating tax rates to a full-stack application using Spring Boot.
