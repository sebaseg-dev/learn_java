# Learning Java – Week 0

Week 0 – Preparing my own learning path

Workload: ~5h

## Context

In January 2026, I earned my RNCP6 (Software Designer and Developer) degree from OpenClassrooms. During the final quarter of 2025, I independently developed a LegalTech SaaS. I leveraged my degree coursework and self-taught PHP/Symfony skills to build a full-stack application featuring a decoupled architecture (Symfony and React connected via a RESTful API) and a PostgreSQL database. Unfortunately, the project was shelved commercially at the end of the year following downturns in my target market.

Initially, I considered maintaining this software as a portfolio piece to showcase my skills to recruiters—viewing a salaried position as the most stable way to transition my career. However, continuing work on a product that will never be launched felt counterproductive. Furthermore, the roles I am most interested in often require a different set of technical skills.

For the past six months, I have felt the need to add a robust, enterprise-grade language (such as Java, Go, or Rust) to complete my stack. That is the journey I am starting today.

## Why Java?

At first, I considered Go or Rust. They are trendy, used in large-scale production, modern, and well-documented.

However, based on the job market data I have been scraping daily for weeks, those languages are rarely mentioned for junior or mid-level roles in my area.

I am looking for a position that aligns with my core values. I consistently see high-quality offers—often requiring more experience than I currently have—specifically seeking Java/Spring developers. Therefore, I have chosen to align my skill set with market demand.

## Methodology

I have a degree from OpenClassrooms. Although the courses aren't perfect, they are completely free, well-organized, and quick to complete. They structure their content into "paths," which is useful for organizing a self-learning curriculum. Based on the [Full-Stack Developer - Java and Angular](https://openclassrooms.com/fr/paths/2460-developpeur-full-stack-java-et-angular) path, I have designed my own as follows:

1. ✅ [Command Line Interface (CLI)](https://openclassrooms.com/fr/courses/6173491-apprenez-a-utiliser-la-ligne-de-commande-dans-un-terminal)
2. ✅ [Algorithm Fundamentals](https://openclassrooms.com/fr/courses/7527306-decouvrez-le-fonctionnement-des-algorithmes)
3. [Java Programming Foundations](https://openclassrooms.com/fr/courses/8383791-apprenez-a-programmer-en-java)
4. [Maintainable Java Code with MVC and SOLID](https://openclassrooms.com/fr/courses/6810956-ecrivez-du-code-java-maintenable-avec-mvc-et-solid)
5. ✅ [REST APIs for Web Projects](https://openclassrooms.com/fr/courses/6573181-adoptez-les-api-rest-pour-vos-projets-web)
6. [Relational Databases with SQL](https://openclassrooms.com/fr/courses/6971126-implementez-vos-bases-de-donnees-relationnelles-avec-sql)
7. [Java Application with Spring Boot](https://openclassrooms.com/fr/courses/6900101-creez-une-application-java-avec-spring-boot)
8. [Testing for Quality Java Applications](https://openclassrooms.com/fr/courses/6100311-testez-votre-code-java-pour-realiser-des-applications-de-qualite)
9. [Debugging Java Applications](https://openclassrooms.com/fr/courses/6692416-debuggez-votre-application-java)
10. [Spring Data & Persistence](https://openclassrooms.com/fr/courses/6982461-utilisez-spring-data-pour-interagir-avec-vos-bases-de-donnees)

My goal is to apply the concepts from these courses to a live project, building an application from scratch to a rough POC (Proof of Concept) by the end of the learning path.

## Project Idea: "Indie"

When I was looking for information on how to start a company in France and how to choose between different legal forms, I was overwhelmed—even with my legal, accounting, and tax background. It is incredibly complex to calculate how the choice of legal form impacts taxes, and official tools sometimes provide incorrect or undocumented results.

I am developing an open-source app to help entrepreneurs navigate these options. The project will evolve from a simple Java terminal program for calculating tax rates to a full-stack application using Spring Boot.
